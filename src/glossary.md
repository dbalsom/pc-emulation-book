# Glossary

## Floppy Disks

Biaxially oriented polyethylene terephthalate
: A flexible and durable polymer, sold under the brand name Mylar

Floppy disk
: A removable magnetic storage medium made from a flexible, coated disk stored inside a protective jacket or shell.

Cookie
: The flexible Mylar disc witin a floppy disk, upon which magnetic signals are encoded.

Shutter
: The spring-loaded, metal cover over the data window on a 3.5" floppy disk, that serves to protect the cookie from dust and damage.

Spindle
: The central axis around which a floppy disk spins. The spindle usually consists of a clamping and centering mechanism.

Clock bit
: A bit written by a disk encoding scheme to provide timing information for recovering the data stream.

Compact disc
: An optical storage disc read by a laser, commonly abbreviated CD.

Cylinder
: A set of tracks at the same radial position across the usable sides of a disk.

Cylinder number
: The numeric identifier for a cylinder or track position on a floppy disk.

Data window
: The opening in a floppy disk jacket or shell that exposes the magnetic surface to the drive heads.

Disc
: A spelling commonly associated with optical storage media such as compact discs.

Disk
: A spelling commonly associated with magnetic or flash storage media.

Double-sided
: A disk or drive format that can use both sides of the magnetic medium.

Drive door
: The latch, lever, or door that closes the drive and clamps the disk in place.

Drive head
: The read/write head in a disk drive that interacts with the magnetic surface.

Encoded
: Transformed into a physical or logical representation suitable for storage or transmission.

Flippy disk
: A disk prepared so it can be turned over and used from the opposite side in a single-sided drive.

Floppy controller
: The controller hardware that commands the floppy drive and converts between host data and disk signals.

Flux reversal
: A change in magnetic polarity on the disk surface.

Flux transition
: Another term for a flux reversal.

Frequency modulation
: A single-density disk encoding scheme that stores a clock transition with each bit cell.

Hard-sectored
: A disk format that marks sector boundaries using physical holes in the disk.

Head
: A magnetic read/write transducer associated with one side of a disk.

Head loading
: The action of bringing the drive head or heads into contact with the disk surface.

Index hole
: A physical hole used by the drive to detect a fixed angular reference point on the disk.

Index position
: The angular reference position indicated by the index hole and index pulse.

Index pulse
: The electrical pulse generated when the index sensor detects the index hole.

Index sensor
: The sensor that detects the index hole, usually by sensing light through the disk.

Longitudinal recording
: Magnetic recording in which the recorded domains are aligned along the direction of disk travel.

Magnetic heads
: The read/write heads that sense or alter magnetic domains on a disk surface.

Microfloppy
: A small floppy disk format, most commonly the rigid-shell 3.5" disk.

Minifloppy
: A term commonly used for the smaller 5.25" floppy disk after the original 8" format.

Modified frequency modulation
: A disk encoding scheme that improves on frequency modulation by omitting predictable clock transitions.

Perpendicular recording
: Magnetic recording in which domains are oriented perpendicular to the disk surface.

Read pulse
: An electrical pulse produced by the drive's read circuitry when it detects a flux transition.

Sector
: A numbered subdivision of a track used as a unit of disk storage.

Sector gap
: Padding or synchronization space between sectors on a track.

Sector ID
: The identifying fields that describe a sector's cylinder, head, sector number, and size.

Sector number
: The per-track number used to identify a sector.

Sector size specifier
: A field in a sector ID that indicates the sector's encoded data size.

Single-sided
: A disk or drive format that uses only one side of the magnetic medium.

Soft-sectored
: A disk format where sector boundaries are defined by data written on the track instead of physical holes.

Spindle motor
: The motor that rotates the disk while the drive reads or writes data.

System 34 format
: The IBM floppy track format associated with the IBM System/34 and later PC-compatible floppy formats.

Track
: A circular path of recorded data at a specific radius on a disk surface.

Track format
: The layout of address marks, data fields, gaps, and synchronization areas on a disk track.

Write-protect notch
: A cutout or opening used by a drive to determine whether a disk should be writable.
